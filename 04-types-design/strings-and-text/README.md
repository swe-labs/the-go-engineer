# Strings and Text Track

This folder is part of the foundations `s04` path and is now validated under the same README contract as the rest of the foundations lessons and exercises.

Recommended learner flow:

1. Start with `ST.1` strings.
2. Continue through `ST.2` formatting, `ST.3` unicode and runes, `ST.4` regex, and `ST.5` text templates.
3. Finish with `ST.6` config parser project.

The public lesson surfaces live directly in this folder.
