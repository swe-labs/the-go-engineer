# Starter — How to use this repository

Use this starter to practice turning repository structure into a small program.

## Task

Open `main.go` and fill in the missing fields so the program can explain:

- what `metadata/` is for
- what `curriculum/` is for
- what `tools/` is for
- what `docs/` is for
- what `dist/` is for

Then compare your answer with `_solution/`.

## Run

```bash
go run ./curriculum/modules/00-orientation/lessons/01-how-to-use-this-repository/_starter
```

The starter is intentionally simple. The goal is to practice naming the responsibility of each major folder.
