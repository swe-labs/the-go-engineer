# 00 How Computers Work

## Mission

Before you can write code to instruct a computer, you need to understand what a computer actually is. This section demystifies the machine. We strip away the magic and show you exactly what happens when code turns into execution.

The goal is to give you a strong **mental model** of the underlying hardware and operating system. 

## Why This Section Exists Now

When beginners learn to code without learning how a computer works, they treat programming like casting spells. If the spell fails, they don't know why. By learning the mechanics of memory, execution, and the terminal *first*, you will write code with confidence and predict how it will behave.
