# 05 Packages & IO

## Purpose

`05 Packages & IO` teaches how Go programs cross package, process, file, and encoding boundaries.

## Who This Is For

- learners who understand types and design decisions
- developers who want more confidence with modules, files, CLI surfaces, and encoded data

## Mental Model

Software becomes useful when it can interact with real inputs and outputs without turning into
plumbing chaos.

## What You Should Learn Here

- modules and package boundaries
- dependency management and versioning
- CLI tool patterns
- encoding and decoding workflows
- filesystem operations and boundary-safe utilities
