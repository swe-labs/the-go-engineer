# 08 Quality & Test

## Purpose

`08 Quality & Test` teaches how to trust and measure engineering work.

## Who This Is For

- learners who can already build meaningful programs
- developers who want stronger testing, benchmarking, and profiling habits

## Mental Model

Quality is not a vibe.
It is evidence that code behaves correctly, is measurable, and can be improved intentionally.

## What You Should Learn Here

- unit and table-driven testing
- HTTP and client test seams
- coverage and benchmark workflows
- CPU and memory profiling
- performance work driven by evidence instead of guesswork
