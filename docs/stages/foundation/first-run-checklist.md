# First-Run Checklist

Use this checklist if you are starting from zero or if setup feels noisy.

## Goal

Get from "I am not sure what I am doing" to "I can run the first lesson and understand what just
happened."

## Checklist

1. Install Go and confirm `go version` works in your terminal.
2. Clone the repository.
3. Change into the repo folder.
4. Run:

```bash
go version
go run ./01-core-foundations/getting-started/2-hello-world
```

5. Open and run:

- [GT.1 installation](../../../01-core-foundations/getting-started/1-installation/)
- [GT.2 hello world](../../../01-core-foundations/getting-started/2-hello-world/)
- [GT.3 how Go works](../../../01-core-foundations/getting-started/3-how-go-works/)
- [GT.4 development environment](../../../01-core-foundations/getting-started/4-dev-environment/)

## What Success Looks Like

You are ready to move forward when:

- the terminal can find `go`
- `go run` works on the first hello-world lesson
- you know where the lesson folders live
- you are no longer scared by `go fmt`, `go build`, or `go test`

## If Something Breaks

Use [workflow-confidence.md](./workflow-confidence.md).
