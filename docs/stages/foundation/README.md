# 0 Foundation Support Docs

This folder supports the beta `0 Foundation` stage.

Use these docs when you need beginner-first help before moving into the language-heavy parts of the
curriculum.

## Use These In This Order

1. [first-run-checklist.md](./first-run-checklist.md)
2. [terminal-and-files.md](./terminal-and-files.md)
3. [workflow-confidence.md](./workflow-confidence.md)

Then move into the current source track:

- [01-foundations/01-getting-started](../../../01-foundations/01-getting-started/)

## What These Docs Are For

- `first-run-checklist.md`: the shortest possible first-run sequence
- `terminal-and-files.md`: mental-model help for files, folders, and the terminal
- `workflow-confidence.md`: common recovery guidance when beginner setup feels fragile

## What These Docs Are Not

They are not a replacement for the source lessons.
They exist to make the learner's first steps calmer and clearer.
