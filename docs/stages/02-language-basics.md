# 02 Language Basics

## Purpose

`02 Language Basics` turns first contact with Go into actual programming fluency.

## Who This Is For

- learners who finished the getting-started path
- transfer learners who want a disciplined Go fundamentals pass

## Mental Model

This stage teaches how values move, how decisions are made, and how collections behave.
It is the stage where small programs stop feeling mysterious.

## What You Should Learn Here

- variables, constants, and basic types
- conditionals and loops
- arrays, slices, maps, and pointer-aware mutation
- the mental models needed before function boundaries become the main focus

## Finish This Stage When

- you can read and write small Go programs without copying line by line
- control flow no longer feels magical
- you can explain the difference between arrays, slices, maps, and pointer-driven mutation
- you can complete the current milestone surfaces honestly
