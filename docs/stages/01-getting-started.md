# 01 Getting Started

## Purpose

`01 Getting Started` is the calm first on-ramp into the repo.

## Who This Is For

- complete beginners
- learners who want a clean install and first-run path
- experienced developers who want a quick environment sanity pass before deeper work

## Mental Model

This stage is about turning a machine into a place where Go code can run on purpose.
We are building confidence with tools, files, the terminal, and the first execution loop.

## What You Should Learn Here

- how to install and verify Go
- how to run a Go program from the terminal
- how files, folders, and packages relate to a runnable lesson
- how to set up an editor and workflow that feel reliable

## Current Source Content

- [01-getting-started](../../01-getting-started/)

## Finish This Stage When

- you can run the getting-started lessons without guessing
- you understand what `go run` is doing at a high level
- your editor and terminal workflow feel stable enough to continue
