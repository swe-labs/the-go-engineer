# Label Taxonomy

Core contribution lanes:

- `good first issue`: suitable onboarding task
- `docs`: documentation-only changes
- `curriculum`: lesson/curriculum registry changes
- `opslane`: flagship implementation work
- `validation`: validator/CI/proof-surface changes
- `security`: security hardening and policy
- `dependencies`: dependency updates (often Dependabot)

Use at least one scope label (`docs`, `curriculum`, `opslane`, or `validation`) and one type label (`security`, `dependencies`, etc.) where relevant.
